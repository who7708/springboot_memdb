SpringBoot整合内嵌式Derby数据库
# 技术特性：  
1、集成数据库（内嵌数据库：derby）   
2、数据库查询（通过mybatis）  
3、支持启动时自动加载数据库脚本  

# 运行说明
1、运行HspApplication  
2、打开浏览器访问http://localhost:8080/blog/query  
页面会显示:`[{"id":1,"title":"花生皮编程博客"}]`
表示能正常从数据库中读取数据。
